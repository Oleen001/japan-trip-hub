## 1. Access จากโตเกียว

### Tobu Railway (แนะนำ — ตรง ไม่ต้องต่อ)
- **SPACIA X** (Limited Express ใหม่ปี 2023): Asakusa → Tobu-Nikko ~1ชม.50น. **0 ต่อ**
  - ค่าโดยสารพื้นฐาน + LE supplement รวม ~¥3,340 one-way (~¥6,680 ไป-กลับ)
  - มาตรฐาน LE fee เริ่ม ¥1,740; standard seat
  - ต้องจองที่นั่งล่วงหน้า (SPACIA X / Revaty / Kegon ทุกขบวน)
- **Revaty / Kegon** (LE รุ่นเก่ากว่า) ราคาใกล้กัน
- ขบวนธรรมดา (rapid/local) ถูกกว่าแต่ต้องต่อ + ช้ากว่า

### JR (เหมาะเฉพาะคนถือ JR Pass)
- Tohoku Shinkansen: Tokyo/Ueno → Utsunomiya (50-60น.) → ต่อ JR Nikko Line → JR Nikko (45น.)
- รวม ~2ชม. + **ต่อ 1 ครั้งที่ Utsunomiya**
- ค่าใช้จ่าย ~¥5,680 one-way (reserved shinkansen) — แพงกว่า Tobu ถ้าไม่มี pass
- **JR Pass ครอบคลุมเส้นนี้ทั้งหมด** (Shinkansen + JR Nikko Line) → ฟรีถ้ามี pass; แต่ JR Pass ไม่ครอบคลุม Tobu

**สรุป:** ไม่มี JR Pass → Tobu SPACIA X ดีกว่าชัดเจน (ตรง เร็ว ถูกกว่า). มี JR Pass active → JR ฟรี
**difficultyFromTokyo: 2** (ตัวเมือง day-trip ง่ายมาก / แต่ถ้าขึ้น Chuzenji+Senjogahara ต้องต่อบัส + วันยาว → ขยับเป็น 3 เชิงเวลา)

### Nikko Pass (Tobu — สำหรับนักท่องเที่ยวต่างชาติเท่านั้น)
- **NIKKO PASS WORLD HERITAGE AREA**: ¥3,000 ผู้ใหญ่ / ¥1,500 เด็ก · valid 2 วัน
  - รวม: ไป-กลับ Asakusa↔Shimo-Imaichi + รถ Tobu Bus ในเขต World Heritage + รถไฟ Tobu ระหว่าง Shimo-Imaichi/Tobu-Nikko/Shin-Fujiwara
  - **ไม่รวม** LE fee (จ่ายเพิ่ม), ค่าเข้าศาลเจ้า, รถไฟ Taiju
  - เหมาะ: เที่ยวแค่ตัวเมือง (Toshogu/Rinnoji/Shinkyo) ไม่ขึ้นเขา
- **NIKKO PASS ALL AREA**: ¥8,000 ผู้ใหญ่ / ¥4,000 เด็ก · valid 4 วัน (ขึ้นราคาเกือบเท่าตัวปี 2025)
  - รวม: ไป-กลับ Asakusa↔Shimo-Imaichi + **รถ Tobu Bus ทุกสายใน Nikko** (รวมขึ้น Chuzenji/Kegon/Yumoto/Senjogahara) + ล่องเรือทะเลสาบ Chuzenji (กลางเม.ย.–30พ.ย.)
  - **ไม่รวม** LE fee
  - เหมาะ: ขึ้นเขา Chuzenji + Senjogahara + Yumoto (บัสแพง คุ้มถ้าไปหลายจุด)
  - บัส Tobu-Nikko → Chuzenji Onsen เดี่ยวๆ ~¥1,250 one-way → ไป-กลับ + Senjogahara เกิน pass เร็ว

แหล่ง: tobu.co.jp/en/ticket/nikko/city.html, /all.html (official)

## 3. ฤดู / Koyo

- **Forecast 2026 (อิงค่าเฉลี่ยระยะยาว):**
  - Oku-Nikko (Chuzenji/Kegon/Irohazaka บน) เริ่มเปลี่ยน ~5-10 ต.ค.
  - **Lake Chuzenji + Kegon peak ~18 ต.ค. – 1 พ.ย.**
  - Irohazaka: บนสุดเริ่มกลาง-ปลาย ต.ค. → ไล่ลงล่างต้น พ.ย.
  - **ตัวเมือง (Shinkyo/Toshogu) peak ~5-12 พ.ย.** (ช้ากว่าบนเขา)
- โซนสูงต่างระดับ → peak คนละช่วง (เหมือน Tateyama). อยากได้ครบต้องเลือก elevation
- 2025 actual: town center ยังไม่ peak ก่อนต้น พ.ย. (japan-guide koyo report 20 ต.ค. 2025)
- **รถติด Irohazaka หนักมากช่วง autumn**: เสาร์-อาทิตย์/วันหยุด ติดตั้งแต่ ~7 โมงเช้า ยาวถึงบ่าย. แนะนำ weekday + ถึง Nikko ก่อน 7:30 ขึ้นบัสรอบแรก
- ฤดูอื่น: ฤดูใบไม้ผลิ-ต้นฤดูร้อน (มิ.ย. primrose ที่ Chuzenji), ฤดูร้อนหนีร้อนขึ้น Oku-Nikko เย็นสบาย, ฤดูหนาวน้ำตก/onsen เงียบ (บัส Yumoto ลดเที่ยว, ล่องเรือปิด)

## 5. Warnings
1. **รถติด Irohazaka ช่วง koyo** — เสาร์-อาทิตย์ ต.ค.-ต้น พ.ย. ติดตั้งแต่ 7 โมง. ไป weekday + รอบบัสแรก
2. **Akechidaira Ropeway ปิด 16 ม.ค. 2026 – 31 ส.ค. 2027** — อย่าวางแผนขึ้น ช่วงนี้ปิด
3. **ตัวเมือง vs บนเขา peak คนละช่วง** — town ช้ากว่า ~2-3 สัปดาห์. เลือกตามจุดที่อยากได้
4. **NIKKO PASS / LE fee แยกกัน** — pass ไม่รวมค่า limited express, ต้องจ่าย+จองที่นั่งเพิ่ม
5. **ขึ้นเขา = วันยาว** — บัส Chuzenji 50น./เที่ยว + Senjogahara hike 2-3ชม. day-trip ไป-กลับโตเกียวพร้อมขึ้นเขาเหนื่อยมาก → ควรค้าง
6. **ฤดูหนาว** บัส Yumoto ลดเที่ยว, ล่องเรือ Chuzenji ปิด (เปิดกลางเม.ย.-30พ.ย.)
7. **Nikko Pass เฉพาะต่างชาติ** — คนญี่ปุ่นซื้อไม่ได้ (ยกเว้นไปกับต่างชาติ)
