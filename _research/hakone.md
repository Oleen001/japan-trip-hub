## 1. Access — verified

- **Romancecar (ตรง ไม่ต่อรถ):** Shinjuku → Hakone-Yumoto ~85 นาที (japan-guide/Odakyu ระบุ ~80 นาที, schedule จริง 85 นาที) ที่นั่ง reserved ทุกที่
- **Hakone Free Pass ราคา 2025-2026 (verified จาก Odakyu + Hakonenavi):**
  - จากชินจูกุ: 2 วัน **¥7,100** / 3 วัน ¥7,500
  - จาก Odawara (Hakone area only): 2 วัน ¥6,000 / 3 วัน ¥6,400
- **ค่าธรรมเนียม Romancecar (เพิ่มจาก Free Pass):** ¥1,200/เที่ยว · e-ticket ¥1,150
- **รวมไป-กลับจากชินจูกุ:** Free Pass ¥7,100 + Romancecar ไป-กลับ ¥2,400 = **¥9,500**
- **difficultyFromTokyo = 1** — ง่ายสุดในกลุ่มปลายทางวิวฟูจิ ตรง ไม่ต้องต่อรถ ใช้ day-trip ได้สบาย
- **jrPassWorth = false** — Hakone เป็นเครือ Odakyu ไม่ใช่ JR; JR Pass ไม่ครอบคลุม loop (ใช้ JR ได้แค่ถึง Odawara ผ่าน Tokaido Shinkansen)
- **Free Pass คุ้มไหม:** คุ้มแน่ถ้าวน loop ครบ (รถไฟภูเขา+เคเบิล+ropeway ¥2,000-3,000 + เรือ ¥2,220 + บัส = เกิน ¥7,100 ง่ายๆ ในวันเดียว) แหล่ง easytravelfollowme/japantripcost ยืนยันคุ้มสำหรับ 1-day loop

### Hakone loop segment (verified)
Hakone-Yumoto → Tozan train (~40min, switchback) → Gora → cable car (~10min, 1.2km) → Sounzan → ropeway (~4km ข้าม Owakudani) → Togendai → pirate ship (Lake Ashi) → Moto-Hakone → bus → Hakone-Yumoto

## 3. ฤดู — verified

- **ใบไม้เปลี่ยนสีพีค:** กลาง-ปลาย พ.ย. (โซนสูง/Owakudani เปลี่ยนก่อน ปลาย ต.ค.-กลาง พ.ย.; โซนทะเลสาบ/Gora กลาง-ปลาย พ.ย.)
- **เห็นฟูจิชัดสุด:** ฤดูหนาว ธ.ค.-ก.พ. ฟ้าใส อากาศแห้ง ฟูจิยอดหิมะ · ต.ค. เห็นได้ ~ครึ่งหนึ่งของวัน และฟูจิยังไม่มี snow cap
- **เช้า > บ่าย:** ฟ้ามักใสตอนเช้า โอกาสเห็นฟูจิสูงกว่า
- **ฤดูใบไม้ผลิ:** เม.ย.-พ.ค. ซากุระ Gora Park + ไฮเดรนเยีย
- **เลี่ยง:** ฤดูฝน (มิ.ย.-ก.ค.) ฟูจิแทบไม่ออก เมฆบัง; weekend/พีคใบไม้แดงคนเยอะมาก
